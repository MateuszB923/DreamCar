# WWW_Project_Individual

Projekt indywidualny, który przygotowałem to strona wypożyczalni luksusowych samochodów **DREAMCAR**.
Serwis składa się ze strony głównej(menu), oferty, podstron konkretnych aut, informacji o firmie oraz strony kontaktowej.

### User story - 1 - Intuicyjna nawigacja po stronie (Menu)
Jako użytkownik strony DREAMCAR chcę mieć cały czas widoczne proste menu z głównymi sekcjami serwisu,
aby móc szybko przechodzić między stroną główną, ofertą, informacjami o firmie i kontaktem bez szukania linków.
### User story - 2 - Przeglądanie oferty aut
Jako klient wypożyczalni chcę przeglądać listę dostępnych samochodów z podstawowymi parametrami,
aby wybrać auto, które najbardziej mi odpowiada.
### User story - 3 - Podgląd szczegółów konkretnego auta
Jako klient wypożyczalni chcę zobaczyć szczegółowe informacje o wybranym samochodzie (osiągi, napęd, silnik, cena),
aby ocenić, czy warto go wypożyczyć.
### User story - 4 - Możliwość przejścia do kontaktu / rezerwacji
Jako klient wypożyczalni chcę z poziomu strony auta przejść szybko do formularza kontaktowego / rezerwacji,
aby bez szukania w menu móc zapytać o termin.
### User story - 5 - Kontakt z wypożyczalnią
Jako klient wypożyczalni chcę wypełnić prosty formularz kontaktowy (imię, e-mail, temat, wiadomość),
aby zapytać o dostępność samochodu lub szczegóły oferty.
### User story - 6 - Sprawdzenie lokalizacji firmy
Jako nowy klient chcę zobaczyć na mapie, gdzie znajduje się wypożyczalnia, aby łatwo do niej trafić.
### User story - 7 - Zobaczenie opinii innych klientów
Jako potencjalny klient chcę przeczytać opinie osób, które już korzystały z wypożyczalni, 
aby utwierdzić się, że to dobry wybór.
### User story - 8 - Promowanie konkretnych samochodów
Jako właściciel wypożyczalni chcę mieć osobne podstrony dla wybranych modeli aut,
aby móc je wysyłać klientom jako link do „karty samochodu”.
